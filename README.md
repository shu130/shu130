## 4. プロジェクト業務(顧客先)以外での自社業務経験

### 4-1. SESの運営に関する経験：

#### 4-1-1. プロジェクト管理や生産管理
  ① プロジェクトごとのメンバーの工数管理を担当し、全体の原価率や利益率を確認  
  ② 月次プロジェクト収支資料を作成

#### 4-1-2. 社内勉強会の開催
  ① 自社内での技術共有  
  ② 資格取得に向けた勉強会の開催

#### 4-1-3. 後輩や配下メンバーのサポート
  ① OJT形式でのサポート、メンタル面でのサポート  
  ② 資格取得のサポート


### 4-2. SES営業に関する経験(技術見積もり作成)：

#### 4-2-1. 要件整理
  ① 案件概要書・要求仕様書の受領、不明点のヒアリング  
  ② タスク・スケジュール・技術条件・納品物の明確化

#### 4-2-2. WBS作成・工数見積もり
  ① フェーズ分割（設計/構築/テスト）→ タスク分解 → 工数算出  
  ② リスクバッファ・メンバースキルを考慮した調整

#### 4-2-3. 見積書作成・顧客折衝
  ① 工数・単価からコスト算出、原価率・利益率の確認  
  ② 見積書提示 → フィードバック → 再見積もり対応
